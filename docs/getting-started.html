Getting Started

Set up universal-mcp-toolkit with the smallest possible path first.

1. Install

npx universal-mcp-toolkit list
npx universal-mcp-toolkit install

2. Find your host config

  • Windows: %APPDATA%\Claude\claude_desktop_config.json
  • macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
  • Linux: ~/.config/Claude/claude_desktop_config.json

3. Generate config

npx universal-mcp-toolkit config --server github slack filesystem --target claude-desktop
The JSON examples use literal placeholders such as ${GITHUB_TOKEN}. Replace them with real values before pasting into your host config.

4. Validate

npx universal-mcp-toolkit doctor github
npx universal-mcp-toolkit run github --transport stdio
